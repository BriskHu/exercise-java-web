# 连接登录服务器



## 消息码定义

消息码的其他各位代表的含义如下：

| 第一位 | 第二、三位 | 第四位 | 第五、六位 | 用途                 |
| ------ | --------- | ------ | --------- | -------------------- |
| 1      | 00        | 1      | xx        | 参数校验不通过        |
| 1      | 00        | 2      | xx        | 设备基础信息消息码        |
| 3      | 00        | 1      | xx        | 参数校验不通过        |
| 5      | 00        | 1      | xx        | 参数校验不通过        |
| 7      | 00        | 1      | xx        | 设备与连接组件之间的通信消息码    |
| 7      | 00        | 2      | xx        | App服务组件与连接组件之间的通信消息码    |
